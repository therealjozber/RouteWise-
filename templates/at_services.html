AT Dashboard Configuration Checklist
SMS Incoming (required for driver reports via SMS)
  1. Go to AT Dashboard → SMS → Short Codes / Alphanumeric
  2. Set Callback URL: https://yourdomain/at/sms/incoming/
  3. Set Delivery URL: https://yourdomain/at/sms/delivery/
USSD (required for *code# menu)
  1. Go to AT Dashboard → USSD → Create Channel
  2. Set Callback URL: https://yourdomain/at/ussd/
  3. Note your USSD code (e.g. *384*5959#)
  4. For local: use ngrok http 8000
Voice (for outbound Critical alerts)
  1. Go to AT Dashboard → Voice → Phone Numbers
  2. Add AT_VOICE_NUMBER to your .env
  3. Set Voice Callback: https://yourdomain/at/voice/
Environment Variables
AT_USERNAME=sandbox
